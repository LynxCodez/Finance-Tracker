# 💰 Personal Finance Tracker (Python)

A simple command-line finance tracker I built to manage income and expenses, with features like categories, summaries, and data persistence.

I wanted to build something more practical than small scripts, so this project focuses on real-world usage and clean structure.

---

## 🚀 Features

- Add income and expenses  
- Categorize transactions (food, transport, etc.)  
- View balance (income vs expenses)  
- View full transaction history  
- Delete transactions  
- Expense summary by category  
- Data saved using JSON (persists after closing)  

---

## 🛠️ Tech Used

- Python  
- JSON (data storage)  

---

## ▶️ How to Run

```bash
git clone https://github.com/LynxCodez/finance-tracker.git
cd finance-tracker
python main.py
```
